What is a Markup Language?

A Markup Language is a system used to structure content on the web. It uses tags to define elements inside a document.

HTML vs XML vs Markdown

HTML (HyperText Markup Language) is used to create webpages.
XML (Extensible Markup Language) is used to store and transport data.
Markdown is a lightweight markup language used for formatting text.

"HTML structures the web, XML structures data, and Markdown structures simplicity."
